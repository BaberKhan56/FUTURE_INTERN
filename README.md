### Task 3: College Event Feedback Analysis

**Objective:**  
Analyze student feedback from college events to uncover satisfaction trends and suggest improvements using survey data. Apply sentiment analysis to gain actionable insights.

**Skills Gained:**  
- Data cleaning & preprocessing  
- Sentiment analysis using NLP  
- Data visualization (charts & plots)  
- Reporting insights and recommendations

**Tools Used:**  
- Python  
- Google Colab  
- pandas  
- TextBlob / VADER  
- seaborn & matplotlib  

**Deliverables / Folder Structure:**  

Task_3_College_Event_Feedback_Analysis/
├── data/
│ ├── event_feedback.csv
│ └── cleaned_event_feedback.csv
├── notebook/
│ └── College_Event_Feedback_Analysis.ipynb
├── screenshots/
│ ├── sentiment_distribution.png
│ └── rating_vs_sentiment.png
└── README.md


**Insights & Recommendations:**  
- Most feedback is **positive**, showing high satisfaction.  
- Negative sentiment often points to **audio issues, event duration, and lack of interactivity**.  
- Ratings correlate well with sentiment, validating results.  
- Recommendations:
  - Improve audio & venue arrangements  
  - Add more interactive sessions  
  - Optimize event duration  
  - Collect feedback regularly
