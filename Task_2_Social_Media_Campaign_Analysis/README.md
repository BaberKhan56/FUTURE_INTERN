#  Task 2 – Social Media Campaign Performance Tracker

##  Project Overview
This project focuses on analyzing **Facebook and Instagram ad campaign data** to evaluate marketing performance and provide actionable insights. The analysis measures key metrics such as **Engagement, CTR (Click-Through Rate), CPC (Cost Per Click), and ROI (Return on Investment)** and presents them through an interactive dashboard.

The goal is to help marketing teams understand which campaigns perform best and how advertising budgets can be optimized.

---

##  Objective
- Analyze social media advertising data
- Evaluate campaign performance and user engagement
- Compare Facebook vs Instagram effectiveness
- Measure CTR, CPC, and ROI
- Build a dashboard for data-driven decision making

---

##  Tools & Technologies Used
- **Microsoft Excel – Data cleaning & calculations  
- **Power BI  – Dashboard creation & visualization   
- **GitHub** – Project version control and hosting  

---

##  Key Metrics Analyzed
- **Impressions** – Number of times ads were shown  
- **Clicks** – Number of ad clicks  
- **CTR (%)** – Clicks ÷ Impressions × 100  
- **CPC** – Spend ÷ Clicks  
- **Engagement** – Likes, comments, shares  
- **Conversions** – Purchases or sign-ups  
- **ROI** – (Revenue − Spend) ÷ Spend  

---

##  Analysis Process
1. Collected and prepared raw ad data  
2. Cleaned data and handled missing values  
3. Calculated CTR, CPC, and ROI using Excel formulas  
4. Performed pivot-based analysis by campaign and       platform  
5. Imported cleaned data into Power BI  
6. Built an interactive dashboard with filters and KPIs  

---

##  Dashboard Features
- KPI cards for Spend, Revenue, CTR, ROI  
- Campaign-wise performance comparison  
- Platform-wise (Facebook vs Instagram) analysis  
- Trend analysis over time  
- Interactive filters for campaigns and dates  

---

##  Key Insights
- **Flash Sale campaigns generated the highest ROI**
- **Instagram ads showed higher CTR compared to Facebook**
- **High spend does not always guarantee high revenue**
- **Brand Awareness campaigns had lower conversion efficiency**
